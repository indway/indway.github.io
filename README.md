# Indway

> *An experimental personal lab where ideas, AI, and experiences collide to solve real-world problems pragmatically.*

Welcome to the **Indway** public repository. This isn't just a shelf of code libraries or boilerplate templates for you to clone and go. Instead, this is the living documentation of my digital experimentation journey.

I built this lab as an ecosystem where I:
1. Document raw insights and unfiltered, honest opinions.
2. Build AI-powered utilities, custom scripts, and semi-automated publishing systems—simply because "existing systems are too complex for my simple needs."
3. Incubate digital products, including Custom CMS integrations (like *JadiCMS*) and various personal productivity tools.

## What You'll Find Here

If you dig into the code or the contents of this repository, you might discover a few useful things:
- **Notes & Diary**: My daily thought processes on user psychology, cognitive biases, copywriting approaches, and reflections on the struggles of being a solo creator and developer.
- **Workflow Mechanics**: Automation scripts that transform raw Markdown files into a static website alongside auto-generated metadata (Sitemaps, RSS, JSON)—all powered by custom CLI-based AI content generators.
- **Digital Products & Directory**: The seeds of pragmatic tools built the moment they were needed. They are functional, raw, and sometimes actively defy conventions (embracing anti-perfectionism).

## The Philosophy

*Start where you are. Use what you have. Do what you can.*  
There are no rigid structural rules here. If a solution looks technically dirty (nerd hacker-style) but perfectly solves the immediate problem at hand, it will be proudly deployed without hesitation.
