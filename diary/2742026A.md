# Ceritanya

ya kemarin saya ada kembangkan sebuah pendekatan baru bagi saya dalam pengembangan CMS, dengan git-based style.

kenapa pendekatan ini menarik?
Ya saya pakai untuk website sekolah awalnya saya setup dengan decap untuk ekosistem astro.js dan saya lihat ada yang kurang menarik atau saya butuh sesuatu yang tidak ada disana dan akhirnya saya kembangkan sendiri dengan nama produk "JadiCMS".

Pendekatannya mirip git-cms tapi dengan konfigurasi beberapa tipikal data saya buat support jenis json dan juga media untuk terhubung ke cdn.

kita tahu bahwa git based apalagi untuk gambar atau media, reponya itu sangat-sangat terbatas kan.

Nah itu yang akhirnya membuat pendekatan baru dimana kalau upload juga otomatis compress ke webp atau avif.

ini menarik, karena tulisan yang saya  buat ini menggunakan pendekatan yang sama seperti jadiCMS tapi lebih simple dan sedikit nerd atau hacker style.

Sambil dijalan bisa update konten dari isi pikiran dan automtis publish ke web ditambah automation pakai git action untuk generate sitemap dan stuff.
